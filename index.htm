How does it work?

So far I've got this list of common prefixes that can be hyphenated

Any text that begins with these prefixes hyphenated will be counted as one word.

Formatted numbers count as one word: 

1.1
3.14159
12345.6789
Â£1,000,000.00
$1.000,99

Dates in numeric format count as one word: 

1/2/19
1/02/2019
21-12-2019
2019-11-04

Times in numeric format count as one word: 

23:59
00:00:01

Once all the special cases have been converted to single words, then all the remaining punctuation is replaced with a space. The words are then counted using space as the delimiter between words.
